# E-commerce Sales Dashboard

## Project Overview
This project analyzes e-commerce sales performance using Power BI, Python Pandas, and SQL.

## Tools Used
- Power BI
- Python
- Pandas
- SQL

## Key Insights
- Technology generated the highest sales.
- West region showed strong performance.
- New York City generated the highest revenue.
- Sales distribution varies significantly across categories.

## Dashboard Features
- KPI Cards
- Interactive Slicers
- Sales by City
- Sales by Region
- Sales by Category

## Files Included
- Power BI dashboard (.pbix)
- Python notebook
- SQL queries
- Dashboard screenshots
